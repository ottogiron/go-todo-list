# Go Todo List
Implements a todo list using <a href="http://emberjs.com/">Ember</a> and Chapi <a href="https://github.com/ottogiron/chapi">Chapi</a>


## Development

### Run the server

```bash
  cd $GOPATH/github.com/ottogiron
  git clone https://github.com/ottogiron/gotodo.git
  cd gotodo
  go get
  go run server.go
```
